Selected Recent Projects

  • 2004-2010
    CAREER: Towards Unified Theory of Wireless Communications
    NSF
    • This is an NSF early career award that helped us to establish the theoretical foundation of the modern wireless physical-layer communication theory.
  • 2007-2011
    Information Theory for Mobile Adhoc Networks (ITMANET)
    DARPA
    • This is a large joint program between Stanford, MIT, Caltech, and UIUC, led by Prof. Andrea Goldsmith of Staford, to develop the theory for network communications.
  • 2012 - 2017
    Information Geometry and Applications in Data Science
    AFOSR
    • This is a project on fundamental theoretic development of new tools of mathematical analysis that is aimed at broadening the application of information theory to topics on learning and data science
  • 2020-2023
    MLWiNS: Deep Neural Networks Meet Physical Layer Communications -- Learning with Knowledge of Structure
    NSF / Intel
    • This is a joint project to tudy the use of machine learning techniques in wireless communication systems. Our project was chosen as the "highlighted" project at the end for its direct impacts on industrial practice.
  • 2019 - 2024
    MURI: Science of Tracking, Control and Optimization of Information Latency for Dynamic Military IoT Systems
    ONR
    • This is a large joint project with Virginia Tech. on applying new tools in data science for better control of large data networks, to improve the latency property and security of communications.
  • 2022 - 2026
    HAYSTAC: Hidden Activity Signal and Trajectory Anomaly Charactorization
    IRAPA
    • This is a collaborative project with industrial partner STR Inc. to simulate and detect abnormal behaviors in ultra-large user-trajectory datasets.
  • 2023-2027
    Learning-based ORAN Testing
    National Telecommunications and Information Administration (NTIA)
    • This recent project is a part of the Biden administration's wireless innovation funds. The goal is to develop machine-learning-based ORAN development tools.
